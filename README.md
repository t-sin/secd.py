# secdm

SECD machine toys.

## How to run

Clone it, 

```sh
$ git clone https://github.com/t-sin/secdm.git
$ cd secdm
```

and run!

```
# with interpreter
$ python3 secdm.py

# from source code
$ python3 secdm.py program/test.s
```
